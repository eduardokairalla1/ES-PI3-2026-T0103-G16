# Frontend — App

Flutter app for Android and iOS.

## Requirements

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (stable channel)
- [FlutterFire CLI](https://firebase.flutter.dev/docs/cli/) — `dart pub global activate flutterfire_cli`
- Android Studio or VS Code with the Flutter extension
- Android/iOS emulator or physical device

## Setup

```bash
flutter pub get
```

Connect the app to Firebase (generates `lib/firebase_options.dart`):

```bash
flutterfire configure
```

## Commands

| Command | Description |
|---|---|
| `flutter run` | Run the app on a connected device/emulator |
| `flutter devices` | List available devices |
| `flutter pub get` | Install dependencies |
| `flutter pub upgrade` | Upgrade dependencies |
| `flutter build apk` | Build Android APK |
| `flutter build ios` | Build iOS app |
| `flutter test` | Run tests |
| `flutter clean` | Clear build cache |

## Project structure

```
app/
├── lib/
│   ├── main.dart               # App entry point
│   └── firebase_options.dart   # Firebase config (generated by FlutterFire CLI)
├── android/                    # Android-specific files
├── ios/                        # iOS-specific files
├── test/                       # Widget and unit tests
└── pubspec.yaml                # Dependencies and app version
```
